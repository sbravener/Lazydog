# Lazy Dog Computing Domination Pack

This package includes a static website designed for GitHub Pages, with:
- Home page
- Services page
- Long-form local service pages
- Vertical pages
- Location pages
- About, Contact, Privacy, and 404 pages
- 30 blog posts
- Sitemap and robots.txt
- Basic schema markup for local business, services, and articles

## Deployment on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files from this package to the repository root.
3. In GitHub, open **Settings > Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save.

## Form handling
The contact page uses a `mailto:` action so the site remains deployable on GitHub Pages without backend code. To capture live submissions, replace the form action with a hosted provider such as Formspree, Basin, or Netlify Forms.
