# Lazy Dog Computing – Full Website Package

This package is structured for direct upload to GitHub.

## Included
- full website
- blog library
- service area landing pages
- geo-focused conversion funnels
- GTM, GA4, and Microsoft Clarity tracking
- updated sitemap.xml and robots.txt

## Folders
- `/assets` – CSS, JavaScript, images
- `/blog` – blog index and all blog articles
- `/verticals` – industry pages
- `/service-areas` – geo-optimized local landing pages

## Upload to GitHub
1. Create or open your GitHub repository.
2. Upload the full contents of this zip to the repository root.
3. Keep the folder structure exactly as provided.
4. If you use GitHub Pages, confirm the publish source is the branch containing these files.
